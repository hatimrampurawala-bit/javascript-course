# javascript

JavaScript was originally created by Brendan Eich in 1995 and it was originally designed to be a scripting language for web browsers. But later JavaScript was turned into a standalone language that can be used directly on your computer.

this lead to JavaScript becoming a popular language for web development and other applications.

## Node.js
Node.js is a JavaScript runtime that allows you to run JavaScript on your computer outside of a web browser. It is built on top of the V8 JavaScript engine and provides a set of libraries for building server-side applications.
